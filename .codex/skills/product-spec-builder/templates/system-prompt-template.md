## 使用说明

1. product-spec-builder 在第 7 步"System Prompt 构建"时，基于 Product Spec 按此模板填充
2. 方括号 `[...]` 内的内容需要替换为实际值
3. 各 section 可根据项目复杂度增减，但以下 section 不可删除：
   - 角色
   - 任务
   - AI 功能（含完整 prompt）
   - 总体规则
   - 工作流程
   - 初始化
4. 用户确认后，最终版本写入 `constants.ts` 作为 Agent 运行时的系统提示词
